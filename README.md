# 🏠 Хаус Мастер — Telegram Bot

Telegram-бот для сервиса бытового ремонта и обслуживания помещений.

## Возможности

- 🔧 Приём заявок на разовый ремонт
- 🔄 Оформление регулярного обслуживания (кафе, офисы, квартиры)
- 📋 История заявок для клиента
- 📞 Кнопка "Перезвоните мне"
- 🔔 Уведомления клиенту при смене статуса
- ⭐ Отзывы после выполнения
- 📢 Рассылка всем пользователям
- ⏰ Напоминание владельцу если заявка висит > 2 часов
- 📅 Еженедельная сводка по понедельникам

## Установка и запуск

### Локально

```bash
pip install -r requirements.txt
python bot.py
```

### Переменные окружения

Перед запуском задай в Railway (или в `.env`):

| Переменная  | Описание                        |
|-------------|---------------------------------|
| `BOT_TOKEN` | Токен от @BotFather             |
| `OWNER_ID`  | Ваш Telegram ID от @userinfobot |

> ⚠️ Никогда не заливай токен в GitHub!

## Деплой на Railway

1. Залей репозиторий на GitHub
2. Зайди на [railway.app](https://railway.app)
3. New Project → Deploy from GitHub repo
4. Выбери репозиторий
5. В разделе Variables добавь `BOT_TOKEN` и `OWNER_ID`
6. Railway запустит бота автоматически

## Команды владельца

| Команда | Описание |
|---------|----------|
| `/orders` | Активные заявки |
| `/stats` | Общая статистика |
| `/week` | Сводка за 7 дней |
| `/status [№] [статус]` | Сменить статус заявки |
| `/broadcast [текст]` | Рассылка всем пользователям |
| `/prices` | Прайс услуг |
| `/contacts` | Контакты |

**Статусы:** `принята` → `в работе` → `выполнено` / `отменена`
